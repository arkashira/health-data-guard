# Health Data Guard
A Python module for encrypting and decrypting health data while ensuring compliance with state-level regulations.

## Usage
1. Create an instance of the `HealthDataGuard` class.
2. Use the `encrypt_data` method to encrypt sensitive data with a secret key.
3. Use the `verify_compliance` method to check if a state is compliant with regulations.
4. Use the `decrypt_data` method to decrypt encrypted data with the same secret key.

## Testing
Run the tests using `python -m pytest` in the `tests` directory.
