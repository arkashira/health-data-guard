# Partner Targets
## Introduction
To enhance the value proposition of health-data-guard, we will integrate with key SaaS/API providers in the healthcare and education sectors. This document outlines our partner integration roadmap, highlighting 7 strategic partners, their free-tier limits, integration effort, and the specific user jobs they solve.

## Partner Integration Roadmap

1. **Google Cloud Healthcare API**
	* Free-tier limit: 100,000 requests per month
	* Integration effort: Medium (M)
	* Value-add: Simplifies the integration of electronic health records (EHRs) and streamlines clinical data exchange, solving the user job of "securely exchanging patient data"
	* Affiliate/revenue-share: Yes, through Google Cloud Partner Advantage Program
2. **Microsoft Azure Health Bot**
	* Free-tier limit: 10,000 messages per month
	* Integration effort: Large (L)
	* Value-add: Enhances patient engagement and provides personalized health recommendations, solving the user job of "improving patient outcomes"
	* Affiliate/revenue-share: Yes, through Microsoft Azure Marketplace
3. **Canvas LMS API**
	* Free-tier limit: 100 requests per day
	* Integration effort: Small (S)
	* Value-add: Enables secure and compliant exchange of student data, solving the user job of "streamlining student information management"
	* Affiliate/revenue-share: No
4. **Box API**
	* Free-tier limit: 10,000 requests per day
	* Integration effort: Medium (M)
	* Value-add: Provides secure file sharing and collaboration, solving the user job of "protecting sensitive documents"
	* Affiliate/revenue-share: Yes, through Box Partner Program
5. **Salesforce Health Cloud API**
	* Free-tier limit: 1,000 requests per day
	* Integration effort: Large (L)
	* Value-add: Enhances patient relationship management and provides personalized care, solving the user job of "improving patient relationships"
	* Affiliate/revenue-share: Yes, through Salesforce AppExchange
6. **AWS Comprehend Medical API**
	* Free-tier limit: 10,000 requests per month
	* Integration effort: Medium (M)
	* Value-add: Provides natural language processing for medical text analysis, solving the user job of "extracting insights from medical data"
	* Affiliate/revenue-share: Yes, through AWS Marketplace
7. **OneDrive API**
	* Free-tier limit: 10,000 requests per day
	* Integration effort: Small (S)
	* Value-add: Enables secure file storage and sharing, solving the user job of "protecting sensitive files"
	* Affiliate/revenue-share: Yes, through Microsoft OneDrive Partner Program

## Prioritization
Partners with affiliate or revenue-share programs will be prioritized to maximize revenue potential. The integration effort will be allocated accordingly, with larger efforts dedicated to high-priority partners.
